🪴 ZenGarden – Habit Tracker

ZenGarden is a full-stack MERN-based habit tracking app designed to boost productivity and personal growth. It features a real-time to-do list, habit progress dashboard, and a gamified virtual garden that grows with user consistency.

🧠 Integrated with an AI chatbot for wellness check-ins and motivational quotes. It also includes a speech-to-text journaling system for daily reflections. Uses JWT authentication for secure login and MongoDB for storage.

🛠 Built using Git for version control, Jenkins for CI/CD, and Docker for scalable deployment.

🔋 HCI-Enabled Smart Solar Energy System

This intelligent energy management system predicts solar power generation using machine learning models like LSTM, XGBoost, and Bayesian Optimization. It utilizes real-time weather and sensor data to forecast energy output accurately.

💡 Users can interact with a React UI by inputting weather parameters such as radiation and humidity to receive dynamic predictions and optimization tips.

📊 Model performance is evaluated using R², RMSE, and MAE metrics to ensure precision in smart energy planning.

💊 HealthMate – AI First-Aid Chatbot

HealthMate is a web platform where users log their symptoms and receive instant first-aid suggestions powered by OpenAI’s language model. Designed for quick wellness tracking and emergency help.

🔐 The app uses encrypted MongoDB databases to store user health logs securely, ensuring confidentiality.

📱 Built with a mobile-responsive layout and clean UI to support accessible healthcare for all.
